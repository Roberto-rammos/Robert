¿Por qué tomar la clase?

      Velocidad

  • Es una clas muy interesantela cual exige rapidez para terminar los trabajos en tiempo y forma. Las actividades se realizan a una velocidad adecuada para un mejpor aprendizaje. A mayor capacidad de aprendizaje, mayor velocidad de trabajo o viceversa.

      Mantenimiento

  • Se pretende tener una comunicación constante con la maestra para corregir errores en nuestros códigos si es necesario. Las correciones en los codigos se utilizan para retroalimentar nuestro aprendizaje y si es necesario le de mantemiento al código original.

      Comunicación movil

  • Se trabaja con un punto muy importante, la comunicación entre el grupo, para mejor resultado en los trabajos en el salon de clases. En esta materia se presta a poder utilizar los dispotivos moviles para recaudar información de dicha clase o tema expuesto.
